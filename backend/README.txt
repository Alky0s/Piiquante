PIIQUANTE :

Construction d'une API en backend pour le site HOT TAKES, qui est une galerie de sauces permettant aux utilisateurs de télécharger leurs sauces piquantes préférées, de liker ou disliker les sauces mises en ligne par les utilisateurs.

Le frontend était fourni dans le cadre de ce projet, et a été réalisé avec Angular CLI version 13.2.4.


OUTILS ET LANGAGES UTILISES :

NodeJS, MongoDB, Express


INSTALLATION :

Cloner le repository.

Pour lancer le frontend, se mettre dans le dossier et suivre les instructions données dans le README dédié.

Pour lancer le backend, se mettre dans le dossier puis exécuter les commandes suivantes :

    Pour installer les dépendances :

    > npm install


    Pour démarrer l'API :

    > nodemon server
